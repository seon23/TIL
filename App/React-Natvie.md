## 자유롭게 기록

### 앱 내용 공유
```
const share = () => {
    Share.share({
        message:`${tip.title} \n\n ${tip.desc} \n\n ${tip.image}`,
    });
}
```
>공유할 데이터는 message 키에 준비. Share 도구 꺼냈으면 그 안에 share 함수를 넣어야 한다.<br>

궁금한 점: 이미지 주소(tip.image)를 전달하면 카톡 공유 시 카톡이 알아서 썸네일을 보여준다고 하던데, 실제 전송 시 썸네일이 보이지 않았다.
