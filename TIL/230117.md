## filter 함수
```
const category = (cate) => {
    if (cate == "전체보기"){
        setCateState(state)
    }else {
        setCateState(state.filter((d) => {
            return d.category == cate
        }))
    }
}
```
- 참고로, state에는 data.tip이 들어있다. else일 경우 state(즉, data.tip)을 **순회하여 category가 cate와 같은 딕셔너리를 찾은 후**, 이를 setCateState의 인자로 넘긴다.<br>
